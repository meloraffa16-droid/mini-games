# Changelog — Mini Games

## [2.0.0] — 2026-04-16

### Adicionado
- 🏓 Pong — o classico dos classicos, jogador vs CPU
- 👾 Space Invaders — defenda a Terra contra ondas de aliens
- 🧩 Tetris — encaixe pecas e limpe linhas
- 🎮 Design system 8-BIT com pixel art e Press Start 2P font

### Alterado
- Todos os jogos convertidos para visual 8-bit pixel art
- CSS reescrito: bordas pixel, sombras hard offset, scanlines CRT
- Paleta de cores neon: cyan, yellow, pink, green

## [1.0.0] — 2026-04-15

### Adicionado
- Menu principal com grid de cards responsivo
- Design system completo (tokens, grid, animações)
- Motor base (GameBase) com game loop e delta time
- 🐍 Snake — cobra clássica com controles por teclado
- 🧱 Breakout — destrua blocos com bola e raquete
- ☄️ Asteroids — pilote sua nave e destrua asteroides
